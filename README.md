# Curso_de_Python
Visão geral criada por IA O curso de Python do Curso em Vídeo, ministrado por Gustavo Guanabara, é uma formação gratuita e didática, ideal para iniciantes. Dividido em "Mundos" (1, 2 e 3), o curso aborda desde a lógica básica e instalação até estruturas de dados, módulos e orientação a objetos (POO), com foco prático em exercícios.
